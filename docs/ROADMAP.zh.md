# 路线图

NAVIER-CFD 的长期目标是成为可复现的神经 CFD 模型、数据和实验操作平台。

## 已完成

- 55 个模型的统一目录；
- 52 个可执行原生参考模型；
- 11 个数据集配置；
- 数据集感知 `load_model(..., dataset=...)`；
- 规范化 `CFDSample` 和 `CFDBatch`；
- 统一训练器、检查点和 CFD 指标；
- PIBERT 原生实现；
- 论文证据感知推荐；
- 浏览器推荐器；
- 英文和简体中文技术文档。

## 近期目标

### 官方实现适配

为 PICT、diffSPH 和 NeuralDEM 建立经过验证的专业运行时适配器，并继续核对其他模型与作者仓库的配置差异。

### 数据集加载器

- 官方下载和修订固定；
- 明确的字段映射；
- 几何、边界、法向和表面面积；
- 严格的几何与工况留出；
- 数据校验和与缓存管理。

### 统一基准

- 相同数据划分和归一化；
- 相同训练预算；
- 多随机种子；
- 场、谱、守恒和积分量指标；
- 训练成本、推理时间和盈亏平衡分析；
- 失效案例可视化。

### 模型权重与模型卡

仅在许可证允许时发布经过验证的权重，并记录训练数据、代码提交、超参数、限制和适用范围。

## 中期目标

- 分布式训练和多 GPU；
- 自动超参数搜索；
- 更完整的非结构网格与粒子批处理；
- RANS、LES、多相和 DEM 专用指标；
- 代理模型与数值求解器耦合；
- 自动生成可审计的基准报告。

## 发布原则

任何“复现论文性能”的声明都必须附带：

1. 官方代码或明确的参考实现说明；
2. 数据集版本与正式划分；
3. 完整配置和随机种子；
4. 至少一次独立测试；
5. 指标定义和物理单位；
6. 失败案例和限制；
7. 许可证与引用信息。
