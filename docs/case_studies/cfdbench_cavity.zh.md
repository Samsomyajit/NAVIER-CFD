# CFDBench 方腔流案例

本案例评估模型在不同边界条件和物性参数下重建二维方腔流速度与压力场的能力。

## 推荐模型

FNO、PINO、U-NO、PIBERT 和 DeepONet。

```python
model = load_model("fno", dataset="cfdbench", sample=sample)
```

## 数据划分

训练、验证和测试应按雷诺数、盖板速度或其他工况参数分组。若目标是外推，应将完整参数区间留出，而不是随机混合相邻工况。

## 指标

- 速度和压力 RMSE/MAE；
- 相对 \(L_2\)；
- 散度 RMS；
- 中心线速度剖面；
- 主涡和次级涡位置；
- 推理时间。

## 注意事项

边界节点在总体损失中占比较小，但对物理解读非常重要。建议单独报告壁面和内部流体区域误差，并保留无量纲化定义。
