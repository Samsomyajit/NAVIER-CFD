# CFDBench 圆柱绕流案例

本案例研究圆柱尾迹中的速度、压力和涡脱落预测。

## 推荐模型

FNO、PINO、PIBERT、Transolver（点集转换后）和时序算子模型。

## 实验设置

- 使用连续历史窗口预测未来场；
- 按雷诺数或时间段严格留出；
- 避免相邻时间帧同时进入训练和测试；
- 对流体域和圆柱边界使用明确掩码。

```python
model = load_model("pibert", dataset="cfdbench", sample=sample)
```

## 指标

- 速度、压力场误差；
- 相关性和余弦相似度；
- 涡量与频谱误差；
- Strouhal 频率偏差；
- 长时间滚动误差；
- 若有表面信息，报告阻力和升力系数。

模型可能在整体场指标上表现良好，但在圆柱近壁和剪切层区域出现局部大误差，因此必须提供空间误差图和最差案例。
