# RealPDEBench 圆柱案例

本案例评估模型从仿真圆柱流动迁移到真实实验或观测域的能力。

## 推荐模型

PIBERT、PINO、FNO、DeepONet、Conformalized-DeepONet 和 TANTE 风格自适应模型。

## 对比协议

1. 仿真训练、仿真测试；
2. 仿真训练、真实测试；
3. 少量真实数据微调；
4. 测试时自适应；
5. 不确定性与校准。

## 关键指标

- 仿真域和真实域分别报告 RMSE/MAE；
- 相位与频率误差；
- 长时间滚动；
- 真实域置信区间覆盖率；
- 迁移前后性能差值；
- 最差时间窗口和局部区域。

数据时间对齐、空间插值和传感器坐标必须固定并写入实验清单。不要把插值误差隐藏在模型误差中。
