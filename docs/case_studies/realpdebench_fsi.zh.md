# RealPDEBench 流固耦合案例

本案例关注流体场与结构位移或变形的联合预测。

## 建模选择

- 多输入算子：MIONet、Fourier-MIONet；
- 几何/坐标模型：GINO、Transolver；
- 物理信息模型：PINO、PIBERT；
- 多物理耦合参考：DeepM&Mnet。

## 输入与输出

输入可包括流体历史场、结构状态、边界条件和材料参数；输出应分别记录流体变量与结构响应。

## 评价

- 流体速度和压力误差；
- 结构位移与速度误差；
- 界面连续性；
- 相位和共振频率；
- 长时间稳定性；
- 能量交换一致性。

训练和测试划分应按结构参数、来流工况或实验批次分组。对于不同网格，必须明确流体—结构界面映射。
