# 汽车三维空气动力学案例

本案例面向 DrivAerNet++、DrivAerML 或 ShapeNet-Car 等变化车辆几何上的三维场和空气动力学量预测。

## 推荐模型

GINO、Transolver、UPT、P3D、AeroTransformer、DoMINO 参考实现和几何图网络。

## 数据配置

- 表示：点云或非结构网格；
- 坐标维度：3；
- 输入：几何、来流和运行参数；
- 输出：表面压力、速度场、阻力或其他积分量；
- 批处理：可变节点数和有效性掩码。

```python
model = load_model("gino", dataset="drivaerml", sample=sample)
```

## 严格几何留出

测试集合应包含训练中完全未见的车辆几何家族。几何近重复、不同网格分辨率或轻微形变不应跨越训练和测试集合。

## 指标

- 表面压力和体场误差；
- 阻力系数及排序相关性；
- 高梯度和尾流区域误差；
- 点数/网格分辨率迁移；
- 显存、吞吐量和端到端推理时间。

三维结果应提供表面图、切片、尾流剖面和最差几何案例，而不仅是单一总体分数。
