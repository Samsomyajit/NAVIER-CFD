# Documentation deployment

NAVIER-CFD documentation is published as a GitHub Pages project site.

- Canonical project URL: `https://samchakraborty.me/NAVIER-CFD/`
- GitHub Pages fallback URL: `https://samsomyajit.github.io/NAVIER-CFD/`

Because `samchakraborty.me` is the custom domain of the account-level GitHub Pages site, GitHub serves project repositories beneath the repository path after the project Pages deployment is active.
