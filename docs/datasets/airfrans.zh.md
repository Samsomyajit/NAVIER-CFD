# AirfRANS 数据集卡

AirfRANS 是二维翼型 RANS 数据集，面向变化几何和运行条件下的空气动力学场预测。

## NAVIER-CFD 默认配置

- 表示：点云或非结构网格节点；
- 维度：2D；
- 几何模式：变化几何；
- 典型目标：速度、压力、湍流相关场和表面量；
- 适用模型：Transolver、GINO、Geo-FNO、MeshGraphNets、GNOT 和几何 Transformer。

```python
model = load_model("transolver", dataset="airfrans", sample=sample)
```

## 必要输入

建议保留：

- 节点坐标；
- 流体域和边界掩码；
- 翼型表面节点；
- 来流速度、攻角和雷诺数；
- 表面法向和面积权重（计算积分量时）；
- 变量的物理单位与归一化方式。

## 严格几何泛化

测试集合应按翼型几何分组留出，而不是随机划分节点或样本。应分别报告场误差、表面压力、阻力/升力误差以及不同攻角和雷诺数下的稳定性。

对于可变点数批处理，必须使用有效点掩码，避免填充值影响损失与指标。
