# CFDBench 数据集卡

CFDBench 是面向二维 CFD 代理建模的基准集合，包含方腔流、管流、溃坝和圆柱绕流等场景，并考虑边界条件、物性参数和几何变化。

## NAVIER-CFD 默认配置

- 表示：结构化网格；
- 默认维度：2D；
- 典型输入：历史速度、压力、几何或工况参数；
- 典型输出：未来或稳态速度与压力场；
- 适用模型：FNO、PINO、U-NO、PIBERT、DeepONet 和结构化 Transformer。

```python
model = load_model("fno", dataset="cfdbench")
```

## 适配建议

实际文件版本可能使用不同字段名。应显式指定输入、目标、坐标和掩码键，并保存通道顺序与归一化统计量。

## 严格评价

建议分别报告：

- 已见几何和已见工况；
- 未见边界条件；
- 未见物性参数；
- 未见几何；
- 长时间滚动误差；
- 速度、压力、散度和谱指标。

不要将相邻时间帧随机分到训练和测试集合中，从而产生时间泄漏。
