# AirfRANS

Steady 2D RANS airfoil dataset with force and boundary-layer evaluation.

**Registry ID:** `airfrans`  
**Tasks:** surrogate, geometry, design, OOD  
**Scenarios:** airfoil interpolation, Reynolds extrapolation, angle-of-attack extrapolation

## Access
Upstream source: https://github.com/Extrality/AirfRANS

## Notes
Report lift, drag, wall shear, boundary-layer metrics, official split, units, mesh statistics, and solver metadata.
