# RealPDEBench 数据集卡

RealPDEBench 用于研究仿真数据与真实实验或观测数据之间的差异，覆盖圆柱绕流、受控圆柱、流固耦合、翼型和燃烧等时空系统。

## NAVIER-CFD 默认配置

- 表示：二维结构化场；
- 时间模式：自回归预测；
- 主要任务：仿真到真实迁移、长时间预测、受控动力学；
- 适用模型：PIBERT、PINO、FNO、DeepONet、时序 Transformer 和不确定性模型。

```python
model = load_model("pibert", dataset="realpdebench", sample=sample)
```

## 关键实验设计

应将仿真域和真实域分别报告，不应只给出混合总体误差。推荐设置：

1. 仅仿真训练与仿真测试；
2. 仿真训练与真实测试；
3. 少量真实数据微调；
4. 跨工况或跨实验批次留出；
5. 不确定性与校准分析。

## 指标

除 RMSE/MAE 外，建议报告相关性、相位误差、长滚动误差、频谱误差、守恒残差和真实域校准质量。

数据配对、时间对齐和传感器位置必须在实验清单中明确记录。
