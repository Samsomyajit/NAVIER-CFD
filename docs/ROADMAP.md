# Roadmap

## 0.1 — contracts and catalogs
Typed specifications, 55-model registry, 11-dataset registry, HF access, CFD recommender, benchmark planner, metrics, case studies, tests, and documentation.

## 0.2 — executable baselines
Native FNO, U-Net, DeepONet, transformer baselines; data converters for PDEBench, CFDBench, and RealPDEBench; unified trainer and evaluator.

## 0.3 — hybrid acceleration
Differentiable-solver adapter protocol, corrector/preconditioner contracts, residual/conservation/fallback API, INC and Solver-in-the-Loop studies.

## 0.4 — geometry and industrial CFD
Unstructured mesh batching, boundary-condition tokens, AirfRANS and DrivAer adapters, Transolver/GINO/UPT/DoMINO runners.

## 1.0 — reproducible public benchmark
Versioned snapshots, license-complete cards, hosted leaderboard, matched budgets, cross-mesh, cross-solver, and sim-to-real tracks.
